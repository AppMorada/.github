# Seja bem-vindo
## O que é o MoradaApp?
O MoradaApp é uma plataforma digital projetada para melhorar a comunicação e a gestão de condomínios, proporcionando benefícios tanto para os moradores quanto para os síndicos. Este projeto oferece uma solução abrangente e conveniente que simplifica a vida em condomínios, promovendo a transparência, a eficiência e a colaboração.

## Repositórios
### Front-end
- [Repositório principal](https://github.com/AppMorada/MoradaApp.Front): repositório central da aplicação de front-end.

### Mobile
- [Repositório principal](https://github.com/AppMorada/MoradaApp.Mobile): repositório central da aplicação mobile.

### Back-end
- [RegistryApi](https://github.com/AppMorada/MoradaApp.registryApi): API dedicada a gerenciar os registros de usuários e condomínios na aplicação;
- [KeyManager](https://github.com/AppMorada/MoradaApp.keyManager): gerenciador de chaves dedicadas a assinatures em processos de criptografia e geração de tokens;
- [AuthGateway](https://github.com/AppMorada/MoradaApp.authGateway): gateway de autenticação dedicado e redirecionar os usuários para os serviços corretamente e de maneira otimizada;
- [EmailSenderFnc](https://github.com/AppMorada/MoradaApp.emailSenderFnc): function dedicada a realizar envios de emails de maneira otimizada (em construção e ainda não implantada).
